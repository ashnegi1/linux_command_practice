# Linux Commands Used

## pwd

Shows the current working directory.

Example

```bash
pwd
```

---

## ls

Lists files and folders.

```bash
ls
```

```bash
ls -l
```

```bash
ls -a
```

---

## cd

Changes directory.

```bash
cd HR
```

```bash
cd ..
```

```bash
cd ~
```
